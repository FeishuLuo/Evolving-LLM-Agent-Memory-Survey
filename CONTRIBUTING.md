# Contributing Guidelines

Thank you for your interest in contributing to this survey! We appreciate help from the community to keep this paper list comprehensive and up-to-date.

## How to Contribute

### Suggesting a New Paper

1. **Open an Issue** with the title `[Paper] <Paper Title>` and include:
   - Paper title and link (preferably arXiv or official publication)
   - Publication venue and year
   - Which category it belongs to (e.g., Storage > Linear > Context Window Adaptation)
   - A one-sentence description of its contribution to agent memory

2. **Or submit a Pull Request** directly:
   - Add the paper entry to the appropriate section in `README.md`
   - Follow the existing table format:
     ```markdown
     | [Paper Title](https://arxiv.org/abs/XXXX.XXXXX) | Venue | Year |
     ```
   - Place it in the correct category according to our three-stage framework

### Reporting Issues

- **Broken links**: Open an issue with the title `[Link] <Paper Title>`
- **Incorrect categorization**: Open an issue with the title `[Category] <Paper Title>` and suggest the correct placement
- **Missing information**: Open an issue describing what's missing

### Proposing New Categories

If you believe a new sub-category is needed, please open an issue with:
- The proposed category name
- Why existing categories don't cover it
- At least 2-3 papers that would belong to it

## Style Guide

- Paper titles should match the original publication exactly
- Use arXiv links when an official publication link is not available
- Venue names should follow standard abbreviations (ACL, EMNLP, NeurIPS, ICML, ICLR, etc.)
- Use `arXiv` for preprints not yet published at a venue

## Code of Conduct

Please be respectful and constructive in all interactions. We are committed to providing a welcoming and inclusive experience for everyone.
